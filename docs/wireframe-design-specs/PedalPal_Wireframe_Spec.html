API Reference
REST endpoints — Laravel 11 · routes/api.php
API Endpoints
MethodEndpointAuthDescription
GET/api/bikes—List all bikes. Optional ?type=beach|mountain
GET/api/bikes/{id}—Single bike detail
POST/api/bikes/{id}/rent—Rent a bike — sets is_available = false
GET/api/accessories—List accessories. Optional ?bikeType=
POST/api/accessories/order—Place order. Bundle discount auto-applied.
POST/api/auth/login—Admin login → session cookie (Sanctum)
POST/api/auth/logout🔒End admin session
POST/api/admin/reset🔒Reset all inventory to defaults
GET/api/admin/logs🔒Recent rental activity log
PATCH/api/admin/bikes/{id}/return🔒Mark bike as returned
Standard response envelope
// Success
{ "success": true,  "data": { ... },   "message": "Bike rented successfully." }

// Error
{ "success": false, "data": null,      "message": "Bike is not available."    }
